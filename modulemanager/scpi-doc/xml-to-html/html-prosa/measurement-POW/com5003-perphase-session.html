Each power module provides four values. Values with numbered indices are phase values and value with index ending 'S' is a sum value.
Four power modules are available:

  • Power 1 module provides power (active/reactive/apparent) values for phase 1. This module is connected to frequency output fout1.
  • Power 2 module provides power (active/reactive/apparent) values for phase 2. This module is connected to frequency output fout2.
  • Power 3 module provides power (active/reactive/apparent) values for phase 3. This module is connected to frequency output fout3.
  • Power 4 module provides power values configurable for all phases. This module is connected to frequency output fout4.
