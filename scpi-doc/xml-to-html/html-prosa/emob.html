These are the commands to be used with EMOB suitcases.

  • EMOB lock state query: It is recommended to use this query only when measurement is not running.

  • If there is only one EMOB controller connected to the device, then parameter to EMOB SCPI commands is optional. When no parameter is specified, device will automatically work with the connected EMOB controller.

  • If there are multiple EMOB controllers connected to the device, then parameter to EMOB SCPI commands is a must. This parameter indicates to which EMOB controller the SCPI command is directed to.
