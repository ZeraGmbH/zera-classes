Each power module provides four values. Values with numbered indices are phase values and value with index ending 'S' is a sum value.
Four power modules are available:

  • Power 1 module provides active power. MEAS:POW1? gives all of these values. This module is connected to frequency output fout1.
  • Power 2 module provides reactive power. MEAS:POW2? gives all of these values. This module is connected to frequency output fout2.
  • Power 3 module provides apparent power. MEAS:POW3? gives all of these values. This module is connected to frequency output fout3.
  • Power 4 module provides all kind of powers. MEAS:POW4? gives all of these values. This module is connected to frequency output fout4.
