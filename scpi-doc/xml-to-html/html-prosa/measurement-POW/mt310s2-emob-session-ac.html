Each power module provides four values. Values with numbered indices are phase values and value with index ending 'S' is a sum value.
Three power modules are available:

  • Power 1 module provides 4-wired active power. MEAS:POW1? gives all of these values.
  • Power 2 module provides 4-wired reactive power. MEAS:POW2? gives all of these values.
  • Power 3 module provides 4-wired apparent power. MEAS:POW3? gives all of these values.
